# Task 1: Python File Organizer

This project is a Python-based file organizer that automatically sorts files into categorized folders such as Images, Documents, Videos, Music, Archives, and Others.

## Features
- Automatic file categorization
- Handles common file types: .jpg, .jpeg, .png, .pdf, .mp3, .mp4, .zip, etc.
- Creates an `Organized_Files` folder in the given path
- Safe to run multiple times (already moved files remain intact)
- Logs actions in `log.txt`

## How to Run
1. Open terminal / PowerShell and navigate to the project folder.
2. Run:
```bash
python organizer.py
