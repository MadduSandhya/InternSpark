# API Data Fetch Project

This project demonstrates how to fetch and display data from a public API using Python.

## Features
- Fetch data using requests module
- Parse JSON response
- Display user details
- Search users by name

## API Used
https://jsonplaceholder.typicode.com/users

## How to Run
1. Install requests:
   python -m pip install requests
2. Run the script:
   python api_fetch.py
